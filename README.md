# Belajar HTML

Repository ini berisi hasil pembelajaran dasar HTML untuk pemula.

## Tujuan

- Memahami struktur dasar HTML
- Mengenal berbagai tag HTML
- Membuat halaman web sederhana
- Melatih penulisan kode HTML

## Materi yang Dipelajari

### 1. Struktur Dasar HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Belajar HTML</title>
</head>
<body>
    <h1>Halo Dunia</h1>
</body>
</html>
```

### 2. Heading

Digunakan untuk membuat judul.

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
```

### 3. Paragraf

```html
<p>Ini adalah paragraf.</p>
```

### 4. Link

```html
<a href="https://www.google.com">Kunjungi Google</a>
```

### 5. Gambar

```html
<img src="gambar.jpg" alt="Contoh Gambar">
```

### 6. List

List tidak berurutan:

```html
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
```

List berurutan:

```html
<ol>
    <li>Belajar HTML</li>
    <li>Belajar CSS</li>
    <li>Belajar JavaScript</li>
</ol>
```

## Contoh Halaman Sederhana

```html
<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama Saya</title>
</head>
<body>

    <h1>Selamat Datang</h1>

    <p>Ini adalah website sederhana yang dibuat menggunakan HTML.</p>

    <a href="https://github.com">GitHub</a>

</body>
</html>
```

## Kesimpulan

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML menjadi dasar dalam pengembangan website sebelum mempelajari CSS dan JavaScript.
