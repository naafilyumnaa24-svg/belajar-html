# Belajar HTML

Repository ini dibuat untuk mempelajari dasar-dasar HTML.

## Materi yang Dipelajari

- Struktur dasar HTML
- Heading
- Paragraf
- Link
- Gambar
- List

## Contoh Kode

```html
<!DOCTYPE html>
<html>
<head>
    <title>Belajar HTML</title>
</head>
<body>
    <h1>Halo Dunia</h1>
    <p>Saya sedang belajar HTML.</p>
</body>
</html>
```

## Kesimpulan

HTML adalah bahasa markup yang digunakan untuk membuat halaman web.
